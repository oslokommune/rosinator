# rosinator

> ⚠️ Prosjekt foreløbig i PoC/MVP stadie, dokumentasjon kommer etterhvert. ⚠️

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
